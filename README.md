# StreetWatch
This project acts as a way to see the danger level of crime near you or where you wanna go. This will allow you to make decisions with the provided information.

## Features
- Search locations by address
- Display area danger levels
- Interactive map interface
- User accounts and saved locations

## Built With
- Django
- HTML/CSS/JavaScript
- DoorFront API

## Installation
1. Clone the repository
2. Create a virtual environment
3. Install requirements
4. Run the server

## Future Plans
- Filters
- Auto-Complete address finder
